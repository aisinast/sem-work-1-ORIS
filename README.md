# Семестровая работа №1 по "Основам разработки информационных систем"
*Попова Анастасия, 11-401*

#### Суть:
Социальная сеть для обнаружения, оценки и визуального исследования мест. Пользователи делятся фотографиями, обязательно привязывая их к конкретным точкам на карте — от достопримечательностей и ресторанов до баров, парков и скрытых жемчужин города. Платформа решает проблему поиска мест по фотографиям и помогает пользователям открывать новые уголки своего города и мира через визуальный контент.

#### Основные возможности:
*Для гостя:*
* Поиск по названию / локации / пользователям
* Просмотр страницы места со всеми связанными постами и просмотром места на карте
* Безопасная аутентификация

*Для авторизованного пользователя:*
* Загрузка фото с геометкой, фотографией места (обязательно) и тегами (опционально)
* Оценка понравившихся постов
* Создание, редактирование и удаление своих постов
* Настройка личной информации и аватара

#### Технологический стек
Проект выполнен с использованием классического стека Java Web:
*   *Back-end:* Java 17+, Java Servlets, JSP, JDBC
*   *Front-end:* HTML5, CSS3, JavaScript (Yandex Maps API)
*   *Database:* PostgreSQL
*   *Build Tool:* Maven
*   *Web Server:* Apache Tomcat 11
